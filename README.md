# Cloon Talk

I. INTRO:

Cloon Talk Site: https://cloontalk.netlify.app/

This is an App to pull Random Jokes from a Joke API.

II. NOTES:

  1. Built with HTML/CSS/JS/API.
  2. Cloon Talk API - 'https://v2.jokeapi.dev/joke/Any?safe-mode
